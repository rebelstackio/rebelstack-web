  • Rebelstack-built to spec in a few months

    Metrigon uses a RESTful API to allow iOS and Android the app to communicate scores and retrieve new puzzle collections.

    The RESTful interface also permits the custom administration application to post new puzzle collections and retrieve aggregated reports.

    From the outset initial planning to final unit-tested, load-tested, deployed software, Rebelstack built the entire API including user session within 4 months.

  • Rebelstack-built to scale to millions

    The metrigon-gs API scales and diminishes on demand using NodeJS on AWS elastic beanstalk.

    The backend uses postgresql to serve JSON records faster than MongoDB with elaborate relations.

    Postgresql on AWS (RDS) is supported with automated failover warm-standby for continuous uptime.

    Using stampede-resistant caching on the middle-tier, the write-intensive application was load-tested to serve millions of concurrent users with minimal and low-tier hardward - saving money.
